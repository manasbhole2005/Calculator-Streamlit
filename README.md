# 🧮 Calculator (Streamlit)

A basic calculator built using Python and Streamlit that performs fundamental arithmetic operations through a clean web interface.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface
- Instant calculation

## 🛠️ Technologies Used

- Python
- Streamlit

## 📂 Project Structure

```
Calculator/
│
└── calculator.py
```

## ▶️ How to Run

Install Streamlit

```bash
pip install streamlit
```

Run

```bash
streamlit run calculator.py
```
